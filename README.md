# 章魚一號：Tkinter 寵物小遊戲

視窗程式設計 - 期末設計
組員：資工三 S1154036 賴怡勳
資工三 S1154036 楊學鵬

以 Python、Tkinter 與 Pillow 製作的桌面寵物養成遊戲。玩家需要照顧章魚寵物的飢餓、心情與健康，同時透過工作、購物、探索與訓練取得資源或提升能力。


## 功能特色

- 寵物狀態：飢餓、心情、健康、天數與生病狀態
- 玩家狀態：金錢、目前體力與體力上限
- 餵食、洗澡、玩耍、訓練等日常互動
- 商店購買飼料與藥品
- 方向鍵反應工作小遊戲
- 骰子下注小遊戲
- 骰子探索地圖與隨機格子效果
- 閃避鯊魚訓練小遊戲
- GIF 動畫、滑鼠懸停效果與自訂游標
- JSON 存檔、讀檔、新遊戲與舊存檔備份
- 支援 PyInstaller 的資源路徑處理

## 遊戲系統

### 寵物數值

| 數值 | 範圍與用途 |
|---|---|
| 飢餓值 | 0～100，可透過餵食恢復 |
| 心情值 | 0～100，可透過玩耍或部分事件恢復 |
| 健康值 | 0～100，會受到工作、探索等行動影響 |
| 天數 | 睡覺後增加 1 天 |
| 生病狀態 | 健康過低時，外出行動可能使寵物生病 |

健康值低於 20 時，執行商店、工作、賭博或探索等外出行動可能生病。健康越低，生病機率越高。

生病期間：

- 行動消耗的體力加倍
- 體力上限降低 1，最低維持為 1
- 無法玩耍或訓練
- 健康值提高至 40 以上後痊癒

### 玩家資源

- 初始金錢：`500`
- 初始體力：`10 / 10`
- 工作可以獲得金錢
- 商店可以購買飼料與藥品
- 訓練成功可增加體力上限
- 睡覺會恢復體力並自動儲存進度

## 可執行動作

### 餵食

從持有的飼料或藥品中選擇物品，再點擊畫面使用。不同物品會影響飢餓、心情或健康。

### 洗澡

健康值至少為 20 時恢復少量健康；健康過低時洗澡不會產生效果。

### 玩耍

消耗體力並提高心情。寵物生病時無法玩耍。

### 訓練

進入閃避鯊魚的小遊戲，使用方向鍵上下移動章魚。成功通過後，體力上限增加 1。

### 商店

消耗體力進入商店，可使用金錢購買飼料與藥品。

### 工作

進行四回合方向鍵反應遊戲。每回合需在箭頭出現後按下對應方向鍵。

工作收入為：

```text
基本收入 60 + 每次正確輸入 10
```

### 賭博

支付金錢後選擇骰子結果區域，包含：

- 三顆骰子的總和
- 指定三同號
- 大或小

中獎後可選擇繼續下注，下一次下注金額會提高。

### 探索

在環狀地圖上擲骰前進。不同格子可能改變：

- 金錢
- 飢餓
- 心情
- 健康

部分格子會隨機抽取其他事件效果。

### 睡覺

睡覺會：

- 消耗飢餓值與心情值
- 恢復體力
- 天數增加 1
- 自動儲存遊戲

恢復量依睡覺時實際消耗的飢餓值與心情值計算，至少恢復 1 點，最多恢復 5 點體力，且不超過體力上限。

## 操作方式

| 場景 | 操作 |
|---|---|
| 主畫面 | 使用滑鼠選擇功能 |
| 撫摸寵物 | 點擊中央寵物 |
| 餵食 | 選擇物品後，以滑鼠點擊使用 |
| 工作 | 使用 `↑`、`↓`、`←`、`→` |
| 訓練 | 使用 `↑`、`↓` |
| 賭博 | 使用滑鼠選擇下注區域與按鈕 |
| 探索 | 使用滑鼠擲骰與離開 |
| 儲存／載入／新遊戲 | 使用視窗上方的「檔案」選單 |


## 安裝與執行

### 1. 下載專案

```bash
git clone <repository-url>
cd <repository-folder>
```

### 2. 建立虛擬環境

Windows：

```powershell
python -m venv .venv
.venv\Scripts\activate
```

macOS／Linux：

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 3. 安裝套件

```bash
python -m pip install Pillow
```

### 4. 啟動遊戲

請從專案根目錄執行，確保程式能找到 `assets/`：

```bash
python game.py
```
